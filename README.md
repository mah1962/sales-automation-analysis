## Project Overview
Sales automation & analysis project to track revenue, profit, margin,
and customer behavior using Python & Power BI.

## Key Business Questions Answered
- Which product categories generate the highest profit?
- Monthly revenue & margin trends
- Top customers by sales

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Power BI
- Jupyter Notebook

## Automation
- Automated data processing & report generation using Automation.py

## Outputs
- CSV cleaned data
- Charts
- Power BI Dashboard
- Business Report

## Dashboard (Power BI)

An interactive Power BI dashboard was created to visualize sales performance and support business decision-making.

### Dashboard Highlights
- Key KPIs (Total Revenue, Total Profit, Total Orders, Average Order Value)
- Revenue breakdown by City and Category
- Monthly sales trends
- Interactive slicers for Date, City, and Category
- Insights page with business recommendations

### Dashboard File
- Dashboard.pbix

> The dashboard is built on top of the cleaned data generated by the automation script and reflects the same KPIs and insights.

## How to Run
1. Place CSV files inside the data folder
2. Run Automation.py
3. Check output, charts, and report folders
